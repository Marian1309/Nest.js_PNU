# API Documentation: Get Books

Retrieve a list of books with optional pagination.

---

## Endpoint

**GET** `http://localhost:3000/api/books`

## Description

This endpoint retrieves a list of books from the database. Pagination can be controlled using optional query parameters.

## Query Parameters

- **skip** (number, optional): Number of records to skip. Defaults to `0` if not provided.
- **take** (number, optional): Number of records to retrieve. Defaults to `10` if not provided.

## Request Example

```http
GET http://localhost:3000/api/books?skip=0&take=10
```

### Error Responses

The Get Books endpoint may return several error responses based on different failure conditions:

### No Books Found

- **Status Code:** `400 Bad Request`
- **Response Body:**

  ```json
  {
    "message": "No books found"
  }
  ```
