# API Documentation: Delete a Book

Delete a book from the database.

---

## Endpoint

**DELETE** `http://localhost:3000/api/books/${bookId}`

## Description

This endpoint allows you to delete a book by providing the book's `id`.

## Request Body

The body of the request should contain the book details in the following format:

```json
{
  "title": "string",
  "author": "string",
  "description?": "string",
  "price": "number"
}
```

```json
{
  "title": "New Book Title",
  "author": "Author Name",
  "description": "Description of the book",
  "price": 15.99
}
```

## Request Example

```http
DELETE http://localhost:3000/api/books/1
```

## Response

### Success Response

**Status Code:** 200 OK

**Response Body:**

```json
{
  "message": "Book with id 1 successfully deleted",
  "data": {
    "id": 1,
    "title": "The Great Gatsby",
    "author": "F. Scott Fitzgerald",
    "description": "A classic novel set in the Roaring Twenties, following the mysterious Jay Gatsby.",
    "price": 10.99,
    "createdAt": "2024-01-01T12:00:00Z",
    "updatedAt": "2024-01-01T12:00:00Z"
  }
}
```

### Error Responses

The Get Books endpoint may return several error responses based on different failure conditions:

### Not Found

- **Status Code:** `400 Bad Request`
- **Response Body:**

  ```json
  {
    "message": "You can't delete book with id 1 because it not found.",
    "status": 404
  }
  ```
