# API Documentation: Get a specific Book

Retrieve a list of books with optional pagination.

---

## Endpoint

**GET** `http://localhost:3000/api/books/${bookId}`

## Description

This endpoint retrieves a single book based on the provided `id`. If the book is not found, a 404 error message will be returned.

## Query Parameters

- **id** (string, required): The ID of the book to retrieve.

## Request Example

```http
GET http://localhost:3000/api/books/1
```

## Response

### Success Response

**Status Code:** 200 OK

**Response Body:**

```json
{
  "data": {
    "id": 1,
    "title": "The Great Gatsby",
    "author": "F. Scott Fitzgerald",
    "description": "A classic novel set in the Roaring Twenties, following the mysterious Jay Gatsby.",
    "price": 10.99,
    "createdAt": "2024-01-01T12:00:00Z",
    "updatedAt": "2024-01-01T12:00:00Z"
  }
}
```

### Error Responses

The Get Books endpoint may return several error responses based on different failure conditions:

### No Book Found

- **Status Code:** `400 Bad Request`
- **Response Body:**

  ```json
  {
    "message": "Book with id 1 not found.",
    "status": 404
  }
  ```
