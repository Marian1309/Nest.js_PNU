# API Documentation: Add a Book

Add a new book to the database.

---

## Endpoint

**POST** `http://localhost:3000/api/books`

## Description

This endpoint allows you to create a new book by providing the required book details.

## Request Body

The body of the request should contain the book details in the following format:

```json
{
  "title": "string",
  "author": "string",
  "description?": "string",
  "price": "number"
}
```

```json
{
  "title": "New Book Title",
  "author": "Author Name",
  "description": "Description of the book",
  "price": 15.99
}
```

## Request Example

```http
POST http://localhost:3000/api/books
```

## Response

### Success Response

**Status Code:** 200 OK

**Response Body:**

```json
{
  "message": "Book with id 123 successfully created",
  "data": {
    "id": 123,
    "title": "New Book Title",
    "author": "Author Name",
    "description": "Description of the book",
    "price": 15.99,
    "createdAt": "2024-01-01T12:00:00Z",
    "updatedAt": "2024-01-01T12:00:00Z"
  }
}
```

### Error Responses

The Get Books endpoint may return several error responses based on different failure conditions:

### Invalid data

- **Status Code:** `400 Bad Request`
- **Response Body:**

  ```json
  {
    "message": "Invalid Data.",
    "status": 404
  }
  ```
