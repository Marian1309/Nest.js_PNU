# API Documentation: Update a Book

Update a book in the database.

---

## Endpoint

**PATCH** `http://localhost:3000/api/books/${bookId}`

## Description

This endpoint allows you to update a specific book by its `id`. If the book is not found, a 404 error message will be returned.

## Request Body

The body of the request should contain the book details in the following format:

```json
{
  "title?": "string",
  "author?": "string",
  "description?": "string",
  "price?": "number"
}
```

```json
{
  "title": "Updated Book Title",
  "author": "Updated Author Name",
  "description": "Updated description of the book",
  "price": 18.99
}
```

## Request Example

```http
DELETE http://localhost:3000/api/books/1
```

## Response

### Success Response

**Status Code:** 200 OK

**Response Body:**

```json
{
  "message": "Book with id 1 successfully updated",
  "data": {
    "id": 1,
    "title": "Updated Book Title",
    "author": "Updated Author Name",
    "description": "Updated description of the book",
    "price": 18.99,
    "createdAt": "2024-01-01T12:00:00Z",
    "updatedAt": "2024-01-01T12:00:00Z"
  }
}
```

### Error Responses

The Get Books endpoint may return several error responses based on different failure conditions:

### Not Found

- **Status Code:** `400 Bad Request`
- **Response Body:**

  ```json
  {
    "message": "You can't update book with id 1 because it not found.",
    "status": 404
  }
  ```
